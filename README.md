# 🧾 Java Swing Login & Register System

This project is a simple **desktop application** built using **Java Swing** that provides user authentication features such as **registration and login**.

---

## ✨ Features

- User Registration form
- User Login form
- Simple GUI using Java Swing
- Navigation between Login and Register screens

---

## 🛠️ Technologies Used

- Java
- Swing (GUI)
- AWT (for layout and components)

---


---

## 🚀 How to Run the Project

1. Open the project in **IntelliJ IDEA**
2. Make sure JDK is installed
3. Run the `Main.java` file
4. The application window will open

---

## 🔑 Screens Overview

### 📝 Register Screen
- First Name
- Last Name
- Email
- Password
- Confirm Password
- Currency
- Register Button
- Reset Button

### 🔐 Login Screen
- Username
- Password
- Login Button
- Reset Button
